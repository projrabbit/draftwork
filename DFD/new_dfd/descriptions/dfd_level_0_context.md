# DFD Level 0 - Context Overview

Thuật toán: Để tương tác với hệ thống, các tác nhân bên ngoài gửi yêu cầu và nhận kết quả xử lý từ hệ thống.

Bước 1: Tác nhân bên ngoài (Nhân viên, HR/Admin, CEO) gửi yêu cầu hoặc dữ liệu vào hệ thống.

Bước 2: Hệ thống tiếp nhận yêu cầu và xác định chức năng cần thực hiện dựa trên loại yêu cầu và vai trò của tác nhân.

Bước 3: Hệ thống xử lý yêu cầu, bao gồm các nghiệp vụ chấm công, quản lý dữ liệu và tích hợp bản đồ GIS 3D thông qua ArcGIS Platform.

Bước 4: Hệ thống trả về kết quả xử lý, thông báo hoặc dữ liệu phản hồi cho tác nhân tương ứng.

Bước 5: Kết thúc.
